1. What is requests.get() used for?
requests.get() is used to send a GET request to an API or website in order to retrieve data from a server.

2. What is a URL?
A URL is the address of an API or website that a program accesses to get information.

3. What are params?
params are parameters sent with an API request to specify exactly what data is needed, such as location, time range, or data type.

4. What is a response?
A response is the reply returned by the server after it receives and processes a request.

5. What does status_code = 200 mean?
status_code = 200 means the request was successful and the server returned the requested data correctly
